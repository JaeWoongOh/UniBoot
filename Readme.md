# Android Uni-FrameWork boot strap

### Add it in your root build.gradle at the end of repositories
```gragle
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```

### Add the dependency
```gragle
dependencies {
        implementation 'com.github.muabe:UniBoot:v0.5.7'
}
```
